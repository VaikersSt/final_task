Git

  • git clone - клонирование удаленного репозитория.
  • git status - Отображение внесенных изменения в файл.
  • git add - добавляет файл в отслеживаемые.
  • git commit -m "first commit" - делает коммит.
  • git push - вносит изменения на удаленный репозиторий.
  • git log - выводит список всех коммитов
  • git pull - подтягивает изменения из удаленного репозитория в локальный.
